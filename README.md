## 智能编程导师


### Plan

项目名称：智能编程导师（CodeTutorAgent）

问题分析：学习编程的过程中，学习者常面临三大核心痛点：一是缺乏针对性练习，市面上的编程题海量但难度和知识点匹配度低，无法根据个人薄弱环节动态调整；二是代码反馈延迟且质量参差不齐，初学者提交代码后难以即时获得语义级别的评审，只能依赖语法检查或论坛求助，学习效率低下；三是学习路径不清晰，入门者不知从何学起，进阶者容易陷入“不知道下一步学什么”的困境，缺少融合岗位需求与个人能力现状的定制化路径。已有工具多在单点上提供帮助（如OJ判题、语法高亮），缺少一个贯穿“练-评-学”全流程的智能导师。

核心功能：


自适应编程练习引擎：根据用户设定的目标（如掌握Python数据分析、前端开发）和当前能力评估，动态生成难度递进、知识点关联的编程题目，覆盖算法、代码重构、调试等题型。

智能代码Review：对用户提交的代码进行语义级分析，指出逻辑错误、边界情况遗漏、代码坏味道、性能隐患和安全风险；提供具体修改建议、正例对比和关联知识点卡片，生成结构化的Review报告。

个性化学习路径规划：结合用户的知识图谱掌握度、学习风格和目标岗位技能要求，自动生成分阶段学习路径，推荐学习资源（视频、文档、项目），并随学习进度动态调整，设置里程碑与复习提醒。使用长期记忆管理用户的学习进度，并允许可视化展示技能树点亮情况、练习总量、Review评分趋势及行业对标差距，辅助用户感知进步，调整学习策略。

预期成果：交付一个可交互的Web应用原型（亦可封装为Jupyter Notebook或Copilot风格插件），后端集成大语言模型与代码执行沙箱，支持Python、JavaScript等主流语言。用户可进行对话式交互以获取练习、提交代码并获得秒级Review反馈，同时查看动态更新的学习路径看板。最终产出包含完整的使用说明文档、不少于3条真实用户场景的案例演示以及一份可复现的技术报告，展示其在降低学习反馈闭环时间、提升路径合理性方面的量化效果。


### 初步架构

用户界面 (Web / Jupyter)
        │
导师主控Agent (ReAct)
  ├── 调度 ──→ 练习设计师Agent (ReAct + Tools)
  ├── 调度 ──→ 代码审查Agent (ReAct + Tools)
  ├── 调度 ──→ 学习路径规划Agent (Plan-and-Solve)
  └── 调用工具：用户数据、知识图谱、看板分析API
